          Tuboshu v2.2.0 更新说明
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

一、新增功能
────────────────────────────────────────────────

1. 插件市场
   - 新增插件市场功能，支持浏览和安装插件
   - 插件系统优化，设置页面布局同步调整

2. 剪贴板历史记录
   - 完整剪贴板历史记录功能，支持自动轮询记录
   - 置顶 / 收藏 / 全部清空 / 清空下拉菜单
   - 实时客户端搜索（零 IPC 零 SQL）
   - 文本卡片双击复制、单条删除
   - 轮询频率优化（缩短至 1s），重复内容自动去重

3. 站点拖拽排序
   - 站点列表支持拖拽排序，结果实时同步到侧边导航栏


二、重构 & 清理
────────────────────────────────────────────────

1. 数据库引擎替换 — sql.js 🎯
   用 sql.js 替换自研 TuboshuDb 文档数据库，彻底解决自研
   引擎的稳定性与性能瓶颈，为后续功能奠定可靠基础。

2. 浏览器环境模拟模块统一
   将分散的 User-Agent、navigator、WebDriver 等伪造逻辑
   整合为统一模块，插件与伪装层解耦，提升可维护性。

3. 移除遗留功能
   - 清除 winLnk 桌面快捷方式功能（打包配置、迁移代码等）
   - 移除拖放上传死代码（transfer-upload.js）
   - 从 alterRequestHeader 移除 user-agent / accept-language / 域名跳过逻辑

三、Bug 修复
────────────────────────────────────────────────

1. 剪贴板相关
   - 复制历史内容不再触发轮询重复记录
   - 写入前 trim() 归一化，确保查重匹配
   - 消除悬停抖动（opacity/visibility 过渡）
   - const text 无法赋值导致文本检测分支静默失败

2. 浏览器指纹/伪装
   - 多项修复以通过 Google 等网站检测（webdriver/chrome mock/User-Agent 等）
   - engine 级别设置 User-Agent，移除 preload 中 ua/appVersion 覆盖
   - window.chrome 改为增强模式（不替换原生）
   - preload 注入脚本容错处理
   - 修复 userAgentData brands 顺序/拼写

3. 窗口与视图管理
   - 已关闭的 WebContentsView 未从父 View 移除导致孤儿拦截 resize
   - 切换标签页时隐藏的 WebContentsView 未从父 View 移除
   - 拖动窗口缩小后右边框 resize 热区失效

4. 其他
   - 设置页面站点顺序与边栏保持一致
   - 导入站点保留 isOpen 状态
   - Ctrl+B 快捷键改用 globalShortcut
   - 监听器泄漏、索引错误、布尔类型统一等 7 项问题修复


四、版本 & 依赖
────────────────────────────────────────────────

- Electron v36.0.0 → v37.10.3
- 版本号：v2.1.1 → v3.0.0
- 打包体积优化（locales + WebGPU DLL）
- 更新 AI 站点配置，精简站点列表
